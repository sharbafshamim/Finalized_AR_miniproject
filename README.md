# Shamim

An Unreal Engine AR (Augmented Reality) project featuring handheld AR capabilities.

## Project Overview

This is an Unreal Engine project focused on AR functionality, including:
- Handheld AR implementation
- Interactive UI components
- Placeable objects in AR space
- AR session management
- Product asset integration

## Features

- **AR Game Framework**: Custom AR game mode, pawn, and player controller
- **Interactive Placeables**: Objects that can be placed and manipulated in AR space
- **UI System**: Comprehensive UI with menus, buttons, and interactive elements
- **Material System**: Custom materials for AR objects and UI components
- **Android Support**: Configured for Android deployment

## Project Structure

- `Content/HandheldAR/` - Main AR functionality and blueprints
- `Content/ProductAssets/` - Product-related assets and materials
- `Config/` - Project configuration files
- `Shamim.uproject` - Main Unreal Engine project file

## Requirements

- Unreal Engine (version compatible with AR development)
- Android development tools (for mobile deployment)
- AR-capable device for testing

## Getting Started

1. Open `Shamim.uproject` in Unreal Engine
2. Ensure AR plugins are enabled
3. Build and deploy to an AR-capable Android device
